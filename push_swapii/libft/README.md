# Libft

### Description
`Libft` is the first project of the 42 core curriculum. The goal of this project is to recreate a collection of standard C library functions, as well as additional functions that will be used throughout future projects.

This project helps strengthen the understanding of:
- memory management
- strings and arrays
- linked lists
- pointers
- dynamic memory allocation
- makefiles
- libraries in C

The library is compiled into a static archive file called: `libft.a`.

### What is `libft.a`
`libft.a` is a static library generated by compiling the source files of the library project. 

A static library is an archive file (created using the `ar` command) that contains compiled object files (`.o`).

**What it does:** Instead of compiling each `.c` file every time you use a functions, you can link your program directly with `libft.a`. 

This allows you to:
- Reuse your custom functions across multiple projects.
- Avoid recompiling the library source code repeatedly.
- Keep your code organized.
- Recreate / extend parts of the standard C library.

## Instructions
**How to compile:**
1. Clone this repository using:
```
git clone <repo-url>
```
2. Compile the library:
```
make libft.a
```

**How to delete the files:**
- Run `make clean` to clean the object `.o` files.
- Run `make fclean` to clean the object files and `libft.a`.
- Run `make re` to clean and recompile everything.

## Library Content
This library is organized into three parts.

**Part 1 functions** - standard C library functions:
- `ft_isalpha` - checks whether a given character is a letter.
- `ft_isdigit` - checks whether a given character is a digit.
- `ft_isalnum` - checks whether a given character is a letter or a digit.
- `ft_isascii` - checks whether a given value is a valid ASCII character.
- `ft_isprint` - checks whether a given character is printable.
- `ft_strlen` - returns the length of a string.
- `ft_memset` - fills the first 'n' bytes of the memory area pointed to by 's' with the constant byte 'c'.
- `ft_bzero` - sets the first 'n' bytes of the memory area pointed to by 's' to 0.
- `ft_memcpy` - copies 'n' bytes from the memory area pointed to by 'src' to 'dest'.
- `ft_memmove` - copies 'n' bytes from the memory area pointed to by 'src' to 'dest'.
- `ft_strlcpy` - copies a string from 'src' to 'dest'.
- `ft_strlcat` - appends the string 'src' to the end of 'dst' and returns the total length dst + src.
- `ft_toupper` - changes a lowercase letter to uppercase.
- `ft_tolower` - changes an uppercase letter to lowercase.
- `ft_strchr` - locates the first occurrence of 'c' in the string 's'.
- `ft_strrchr` - locates the last occurrence of 'c' in the string 's'.
- `ft_strncmp` - compares at most 'n' characters of two strings and returns the difference when the characters are not the same.
- `ft_memchr` - searches the first 'n' bytes pointed to by 's' for the first occurrence of the character 'c'.
- `ft_memcmp` - compares the first 'n' bytes of two memory areas.
- `ft_strnstr` - locates the first occurrence of the string 'little' in the string 'big'.
- `ft_atoi` - converts the string representation of a number into an integer.
- `ft_calloc` - allocates memory for an array of 'nmemb' elements each of 'size' bytes and initializes all bytes to 0.
- `ft_strdup` - creates a duplicate of the string 's'.

**Part 2 functions** - utility functions:
- `ft_substr` - returns a substring from the string 's'.
- `ft_strjoin` - returns a new string which is the result of the concatenation of 's1' and 's2'.
- `ft_strtrim` - returns a copy of 's1' with the characters specified in 'set' removed from the beginning and the ending of the string.
- `ft_split` - splits the string 's' into an array of strings using the separator 'c'.
- `ft_itoa` - converts an integer to its string representation.
- `ft_strmapi` - applies the function 'f' to each character of the string 's' to create a new string resulting from successive applications of 'f'.
- `ft_striteri` - applies the function 'f' on each character of the string passed as argument.
- `ft_putchar_fd` - outputs the character 'c' to the given file descriptor.
- `ft_putstr_fd` - outputs the string 's' to the given file descriptor.
- `ft_putendl_fd` - outputs the string 's' to the given file descriptor followed by a new line.
- `ft_putnbr_fd` - outputs the integer 'n' to the given string descriptor.

**Part 3 functions** - linked lists functions:
- `ft_lstnew` - returns a new element of the list.
- `ft_lstadd_front` - adds the 'new' element at the beginning of the list.
- `ft_lstsize` - counts the number of elements in a list.
- `ft_lstlast` - returns the last element of a list.
- `ft_lstadd_back` - adds the  'new' element at the end of a list.
- `ft_lstdelone` - deletes the content of a node using the function 'del'.
- `ft_lstclear` - deletes the given element and every successor of that element.
- `ft_lstiter` - iterates the list 'lst' and applies the function 'f' to the content of each node.
- `ft_lstmap` - iterates the list 'lst' and applies the function 'f' to the content of each element.
